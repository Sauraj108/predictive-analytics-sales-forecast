# 📊 Predictive Analytics — Sales Forecasting

## 🚀 Overview
This project uses Machine Learning and Time Series models to forecast future sales.

## 📊 Models Used
- Linear Regression
- Random Forest
- ARIMA (Time Series)

## 📈 Results
- Achieved ~99% accuracy (R² Score)
- Forecasted next 30, 60, 90 days sales

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Statsmodels
- Matplotlib

## 📂 Outputs
- Sales Forecast CSV
- Data Visualizations (PNG charts)

## 💡 Business Insights
- Marketing spend strongly affects sales
- Seasonal trends (festive peaks)
- Mid-year dips identified

pip install pandas numpy matplotlib seaborn scikit-learn statsmodels
